# nomadcoin
Nomadcoin, the awesome coin made in Node.Js

#### repository init
```
yarn init
```
#### crypto JS install
```
yarn add crypto-js
```

#### install package
```
yarn add express morgan body-parser
yarn global add nodemon
```
#### start server
```
yarn dev
```

#### install websocket
```
yarn add ws
```


#### add hex to binary for hash matches
```
yarn add hex-to-binary
```

#### add elliptic
```
yarn add elliptic
```

#### add lodash
```
yarn add lodash
```
